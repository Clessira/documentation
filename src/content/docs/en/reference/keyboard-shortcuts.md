title: Keyboard shortcuts
description: All NowDoing keyboard shortcuts at a glance.
---

NowDoing is designed so you rarely need the mouse. Key shortcuts:

## In the popover

| Action | Shortcut |
|---|---|
| Open / close popover | Left-click the icon |
| Filter activities | Type (focus is in the search field) |
| Move selection up / down | <kbd>↑</kbd> / <kbd>↓</kbd> |
| Confirm selection | <kbd>Enter</kbd> |
| Close without selecting | <kbd>Esc</kbd> |
| Create new activity | <kbd>⌘N</kbd> |
| Open timesheet | <kbd>⌘1</kbd> |
| Switch between views | <kbd>⌘2</kbd> / <kbd>⌘3</kbd> / <kbd>⌘4</kbd> |

## In the prompt (global)

These shortcuts work as soon as the tracking prompt is open — even if another app has focus. You don't need to switch to NowDoing first.

| Action | Shortcut |
|---|---|
| Open the prompt from anywhere | <kbd>⌃⌥⌘L</kbd> |
| Confirm the suggestion | <kbd>⌃⌥⌘J</kbd> |
| Discard the interval | <kbd>⌃⌥⌘O</kbd> |
| Log pinned activity 1–9 | <kbd>⌃⌥⌘1</kbd> … <kbd>⌃⌥⌘9</kbd> |

When the prompt is frontmost, the single-modifier variants also work:

| Action | Shortcut |
|---|---|
| Confirm the suggestion | <kbd>⌃Y</kbd> |
| Discard the interval | <kbd>⌃N</kbd> |
| Snooze (ask again later) | <kbd>⌃S</kbd> |

As long as no text field has focus, you can log pinned activities directly with <kbd>1</kbd>–<kbd>9</kbd>. Type a letter and focus jumps to the search field, filtering your activities — no mouse needed.

The open-prompt shortcut is configurable under **Settings → Keyboard**.

## App menu (NowDoing focused)

| Action | Shortcut |
|---|---|
| Settings | <kbd>⌘,</kbd> |
| Timesheet | <kbd>⌘1</kbd> |
| Quit NowDoing | <kbd>⌘Q</kbd> |

## In the timesheet window

| Action | Shortcut |
|---|---|
| Previous day | <kbd>⌘[</kbd> |
| Next day | <kbd>⌘]</kbd> |
| Today | <kbd>⌘T</kbd> |
| Day / week toggle | <kbd>⌘D</kbd> / <kbd>⌘W</kbd> |
| Export CSV | <kbd>⌘E</kbd> |
